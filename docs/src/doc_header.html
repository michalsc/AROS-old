NEWS

21.2.2000 There is a new support page for AROS for PPC Amiga. It will be the main contact page for people interested in AROS on PPC cards on Amiga hardware and especially those people who are interested in supporting this by working on it.
18.2.2000 The partnership between the Open Amiga Foundation and AROS is now official (Read the press release). OAF will handle the public side (handle publicity) while AROS will concentrate all its energies on development.

To avoid infringing patents, the menu code for AROS has been changed. Menus now look more modern plus they open under the mouse (like a context menu). For the desperate, the old look is still available :-)

Also, Michael Schulz has committed a big chunk of code for AROS i386/native. There is now a VGA.HIDD, AROS can open a workbench screen. There are still some bugs to be ruled out, but it looks very promising.

Lastly for today, Stefan Berger has committed about 20 new applications from fish disks (cfn, aroach, windowtool, touch, shuffle, memomaster, memeter, lyr-o-mat, gooddouble, eval, egypt and memsnap). Some still don't do something useful but they show that porting apps to AROS is really simple.

15.2.2000 A longstanding bug with the dates in the ToC has been fixed (it's not perfect but better than before).
12.2.2000 The autodocs are now here.
11.2.2000 We have begun to move contributed software (for example, examples from Fish disks and Aminet, the small and big games) outside of AROS. These things have now their own graph (that's why the size of AROS became smaller last week) and the code is in AROS-contrib.
8.2.2000 There is progress on the AROS i386/native front. A very basic VGA.HIDD has been written plus some additional stuff (timer, trackdisk, etc).
