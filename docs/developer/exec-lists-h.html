exec/lists.h

Name

exec/lists.h -- exec list handling

Includes

  • exec/nodes.h

struct List

A standard exec list structure. This is used for all kinds of doubly linked lists. Members of this list are of type struct Node.

This structure can be embedded into other structures.

Fields

struct Node * lh_Head

Pointer to the first item in the list. If the list is empty, this points to the lh_Tail field of the list.

struct Node * lh_Tail

Pointer to the last item in the list. If the list is empty, this is NULL.

struct Node * lh_TailPred

This is a fake entry that also points to the last entry in the list. If the list is empty, it points to the list itself, instead.

UBYTE lh_Type

This field specifies the type the nodes in the list. It corresponds to the ln_Type field of the Node structure. See exec/nodes.h for a list of possible types.

UBYTE l_Pad

Unused field for word-padding.

struct MinList

A minimal exec list structure. This is exactly the same as a List structure, except that the lh_Type field is missing. Members of a MinList are of type struct MinNode. Nodes may be used, but you loose all their advantages.

This structure can be embedded into other structures.

Fields

struct MinNode * mlh_Head

Pointer to the first item in the list. If the list is empty, this points to the mlh_Tail field of the list.

struct MinNode * mlh_Tail

Pointer to the last item in the list. If the list is empty, this is NULL.

struct MinNode * mlh_TailPred

This is a fake entry that also points to the last entry in the list. If the list is empty, it points to the list itself, instead.
